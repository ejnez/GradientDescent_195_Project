# GradientDescent_195_Project

For modeling, `richness_per_site` and `ids_per_site` are the two columns we actually fed into the model as features, since they're effort-normalized. Then `species_richness` and `total_identifications` are the raw versions and `n_sites_sampled` is just the denominator. We kept them in the file for transparency and reproducibility but didn't use them as model inputs directly.
